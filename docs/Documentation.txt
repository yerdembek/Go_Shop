Это будет интернет магази как Kaspi.kz или OLX где будет каталог товаров, база данных с зарегестрированными товарами.
Платежная система, профиль, аккаунты. Позиция юзера: покупатель, продовец.
Для продовцов возможность: выкладывать объявление товара, выкладывать фотографий товаров, возможность писать характеристики и описание товара.
Для покупателей возможность: покупать и заказывать товары, выбирать аддресс, выбирать из католога товар, оставлять отзыв на товар.

User - ID, контакты, имя, аккаунт, платежная карта, роль(Seller - *. Buyer - *.).

CartItems - (BuyerID, PostID, Quantity)
Favorites - (BuyerID, PostID)

Post - ID, SellerID, название товара, характеристики, фотографий, описание, цена, дата создания статус.

Review - ID, PostID, содержимое текст, рейтинг звезд, фотография, видео.

Order - ID, BuyerID, Status, дата создания, Address, конечная цена.
